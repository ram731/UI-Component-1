UI-Component

UI-Component is a framework for developing Angular 9 based customer facing web applications as per MyDEQ standers. This project is generated with Angular CLI version 10.1.3.

Development server

Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

Code scaffolding

Run ng generate component component-name to generate a new component. You can also use ng generate directive|pipe|service|class|guard|interface|enum|module.

Build

Run npm run app-local to build the project. The build artifacts will be stored in the dist/ directory. Use the -prod flag for a production build.
